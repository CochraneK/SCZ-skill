# SCZ-skill

精神分裂症领域治疗师&医生视角 Skill，用于 AI 角色扮演场景。

## 简介

SCZ-skill 是一个基于国际/中国临床指南、权威专家思维框架构建的 AI Skill，旨在模拟精神分裂症治疗领域的专业治疗师/精神科医生视角。

## 核心能力

- **症状评估与解释**（阳性/阴性/认知症状）
- **治疗决策咨询**（药物选择、心理干预）
- **医患沟通**（病情解释、建立信任、处理抗拒）
- **康复指导**（社会功能恢复、预防复发）

## 心智模型

1. **症状维度模型** - 理解精神分裂症的多维症状结构
2. **生物-心理-社会模式** - 整合性病因与治疗视角
3. **全病程管理** - 从首发到康复的长期视角
4. **循证决策** - 基于证据的治疗选择
5. **康复导向** - 以功能恢复为核心的治疗目标

## 决策启发式

包含 10 条临床决策启发式，涵盖：
- 药物选择与换药时机
- 维持治疗策略
- 难治性精神分裂症处理
- 心理社会干预时机
- 康复期管理

## 使用场景

- 医学教育与培训
- 患者/家属健康教育
- 初步咨询与信息提供
- 角色扮演练习

## 重要声明

⚠️ **本 Skill 不能替代真实的精神科医生**：
- 不能进行正式诊断
- 不能开具处方
- 不能处理紧急情况
- 信息可能有时效性
- 个体差异大，需个体化评估

如需正式诊疗，请咨询专业精神科医生。
